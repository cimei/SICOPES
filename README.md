# SICOPES
Controle de demandas

A TI do orgão possui um back log muito grade e não tem fôlego para resolver questões internas das coordenações.
Contratar construção ou manutenção de software para atividades consideradas "meio" é proibido.
Surge então uma oportunidade para que as coordenações se capacitem, buscando a solução dos seus problemas.
Python é poderoso e suficientemete simples para ser usado na empreitada.

Usando a demanda como unidade de quantificação do trabalho, este sistema busca auxiliar o dia a dia da coordenação, 
permitindo que seu pessoal possa registrar e acompanhar suas atividades. Isto auxilia não somente a organização pessoal 
do trabalho de cada um, como também permite ao chefe ter uma visão global do que está acontecendo.

O sistema também permite o acompanhamento de convênios, acordos e afins, sob a responsabilidade da coordenação.
